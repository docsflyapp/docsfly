# Tabs Component Example

This page demonstrates how to use the Tabs component in your MDX files.

## Basic Tabs

<Tabs defaultValue="tab1">
  <TabsList>
    <TabsTrigger value="tab1">First Tab</TabsTrigger>
    <TabsTrigger value="tab2">Second Tab</TabsTrigger>
    <TabsTrigger value="tab3">Third Tab</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">
    Content for the first tab. You can put any markdown or React components here.
    
    - List item 1
    - List item 2
    - List item 3
  </TabsContent>
  <TabsContent value="tab2">
    Content for the second tab.
    
    ```javascript
    function example() {
      return "This is a code example in tab 2"
    }
    ```
  </TabsContent>
  <TabsContent value="tab3">
    Content for the third tab with a **bold text** and *italic text*.
  </TabsContent>
</Tabs>

## Code Examples in Tabs

You can organize different programming languages or examples using tabs:

<Tabs defaultValue="javascript">
  <TabsList>
    <TabsTrigger value="javascript">JavaScript</TabsTrigger>
    <TabsTrigger value="python">Python</TabsTrigger>
    <TabsTrigger value="rust">Rust</TabsTrigger>
  </TabsList>
  <TabsContent value="javascript">
    ```javascript
    function greet(name) {
      return `Hello, ${name}!`
    }
    
    console.log(greet('World'))
    ```
  </TabsContent>
  <TabsContent value="python">
    ```python
    def greet(name):
        return f"Hello, {name}!"
    
    print(greet('World'))
    ```
  </TabsContent>
  <TabsContent value="rust">
    ```rust
    fn greet(name: &str) -> String {
        format!("Hello, {}!", name)
    }
    
    fn main() {
        println!("{}", greet("World"));
    }
    ```
  </TabsContent>
</Tabs>
